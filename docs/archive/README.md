# Archive

Historical documents preserved for transparency. Nothing here describes current behavior.

## `historical-bugs/`

Bug reports written **before v1.2.0** (May 12, 2026), when the repo had:
- 6 same-name skill collisions causing install conflicts
- 7 directory/frontmatter name mismatches
- 4 SKILL.md files missing markdown headings

All resolved in v1.2.0. Files kept so anyone reading the project's history (or auditing how it matured) can see the original analyses, not just the changelog.

## Why this folder exists

A previous version of the repo had `BUG_REPORT.md` and `CRITICAL_BUGS_V2.md` in the root, which made first-time visitors think the repo was unstable even after the bugs were fixed. Moving them here (rather than deleting) preserves the audit trail without misleading new users.
